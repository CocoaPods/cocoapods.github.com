Source

The Source class is responsible to manage a collection of podspecs.

The backing store of the podspecs collection is an implementation detail abstracted from the rest of CocoaPods.

The default implementation uses a git repo as a backing store, where the podspecs are namespaced as:

"#SPEC_NAME/#VERSION/#SPEC_NAME.podspec"

172 def search_by_name(query, full_text_search = false)
173   if full_text_search
174     pod_sets.map do |set|
175       begin
176         s = set.specification
177         text = "#{s.name} #{s.authors} #{s.summary} #{s.description}"
178       rescue
179         CoreUI.warn "Skipping `#{set.name}` because the podspec contains errors."
180       end
181       set if text && text.downcase.include?(query.downcase)
182     end.compact
183   else
184     names = pods.select { |pod_name| pod_name.downcase.include?(query.downcase) }
