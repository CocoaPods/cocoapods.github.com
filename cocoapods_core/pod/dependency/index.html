Dependency

The Dependency allows to specify dependencies of a Podfile or a Specification on a Pod. It stores the name of the dependency, version requirements and external sources information.

This class is based on the dependency class of RubyGems and mimics its implementation with adjustments specific to CocoaPods. RubyGems is available under the MIT license.

306 def self.from_string(string)
307   match_data = string.match(/(\S*)( (.*))?/)
308   name = match_data[1]

344 def external_source_description(source)
345   if source.key?(:git)
346     desc =  "`#{source[:git]}`"
347     desc << ", commit `#{source[:commit]}`" if source[:commit]
348     desc << ", branch `#{source[:branch]}`" if source[:branch]
349     desc << ", tag `#{source[:tag]}`"       if source[:tag]
350   elsif source.key?(:hg)
351     desc =  "`#{source[:hg]}`"
352   elsif source.key?(:svn)
353     desc =  "`#{source[:svn]}`"
354   elsif source.key?(:podspec)
355     desc = "`#{source[:podspec]}`"
356   elsif source.key?(:local)
357     desc = "`#{source[:local]}`"
358   else
359     desc = "`#{source.to_s}`"
360   end
361   "from #{desc}"
362 end
