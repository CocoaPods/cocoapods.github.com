TargetDefinition

The TargetDefinition stores the information of a CocoaPods static library. The target definition can be linked with one or more targets of the user project.

Target definitions can be nested and by default inherit the dependencies of the parent.

531 def podspec_path_from_options(options)
532   if path = options[:path]
533     path_with_ext = File.extname(path) == '.podspec' ? path : "#{path}.podspec"
534     path_without_tilde = path_with_ext.gsub('~', ENV['HOME'])
535     file = podfile.defined_in_file.dirname + path_without_tilde
536   elsif name = options[:name]
537     name = File.extname(name) == '.podspec' ? name : "#{name}.podspec"
538     file = podfile.defined_in_file.dirname + name
539   elsif options[:autodetect]
540     file = Pathname.glob(podfile.defined_in_file.dirname + '*.podspec').first
541   end
542 end
