Podfile

The Podfile is a specification that describes the dependencies of the targets of an Xcode project.

It supports its own DSL and generally is stored in files named CocoaPods.podfile or Podfile.

The Podfile creates a hierarchy of target definitions that that store the information of necessary to generate the CocoaPods libraries.

210 def self.from_file(path)
211   path = Pathname.new(path)
212   unless path.exist?
213     raise Informative, "No Podfile exists at path `#{path}`."
214   end
215   string = File.open(path, 'r:utf-8')  { |f| f.read }
216   # Work around for Rubinius incomplete encoding in 1.9 mode
217   if string.respond_to?(:encoding) && string.encoding.name != "UTF-8"
218     string.encode!('UTF-8')
219   end
220 
221   case path.extname
222   when ''
223     Podfile.from_ruby(string, path)
224   when '.yaml', '.cocoapods'
225     Podfile.from_yaml(string, path)
226   else
227     raise Informative, "Unsupported Podfile format `#{path}`."
228   end
229 end
