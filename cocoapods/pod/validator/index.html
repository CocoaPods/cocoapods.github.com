Validator

Validates a Specification.

Extends the Linter from the Core to add additional which require the LocalPod and the Installer.

In detail it checks that the file patterns defined by the user match actually do match at least a file and that the Pod builds, by installing it without integration and building the project with xcodebuild.

352 def parse_xcodebuild_output(output)
353   lines = output.split("\n")
354   selected_lines = lines.select do |l|
355     l.include?('error: ') &&
356       (l !~ /errors? generated\./) && (l !~ /error: \(null\)/)  ||
357       l.include?('warning: ') && (l !~ /warnings? generated\./) ||
358       l.include?('note: ') && (l !~ /expanded from macro/)
359   end
360   selected_lines.map do |l|
361     new = l.gsub(/\/tmp\/CocoaPods\/Lint\/Pods\//,'')
362     new.gsub!(/^ */,' ')
363   end
364 end
